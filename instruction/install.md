# 硬件建议
cpu： 暂无
内存： 根据系统建议4G
硬盘: 建议32G以上  越大越好 如果用于存储数据
网口：一个网口就可以 多网口更好

两个网口 可以实现 一个对内局域网 一个队外局域网
两个网口 纯内网 可以实现 两个不胡同的局域网
两个以上 更多使用场景
